# Construction Tasks Management App

A React + TypeScript app.

## Installation & Setup

- Node.js (v18 or higher)
- npm or yarn

### Installation Steps

1. Clone:

```bash
git clone <repository-url>
cd test-app
```

2. Install:

```bash
npm install
```

3. Start project:

```bash
npm run dev
```

## Time Spent on Features

- Setup & Infrastructure & Exploring RxDB doc (~1.5 hours)
- Authentication System (~1 hour)
- Plan View & Interactions (~3 hours)
- Task Management (~2 hours)
- Checklist System (~2 hours)
